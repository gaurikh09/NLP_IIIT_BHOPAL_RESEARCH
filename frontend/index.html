ℹ️ About This Project

End-to-end Bangla news headline generation using multilingual transformers.

📝 Project Overview

This is an end-to-end NLP application that automatically generates Bangla news headlines from full Bangla news articles using a fine-tuned mT5-small model. The system is trained on the Bangla Newspaper Dataset containing 50,000 article-headline pairs for 3 epochs with 512 token input context on an NVIDIA RTX 4070 GPU.

🛠️ Technology Stack

  • 🐍 Python 3.12
  • 🤗 HuggingFace Transformers 4.x
  • 🔥 PyTorch 2.x + CUDA
  • ⚡ FastAPI + Uvicorn
  • 🌐 HTML5 / CSS3 / JavaScript
  • 📊 ROUGE-Score + NLTK (BLEU)
  • 🖥️ NVIDIA RTX 4070 Laptop GPU
  • 📄 FPDF2 (Report Generation)

🔄 How It Works

The model uses the text-to-text paradigm:

Input: headline: <full_article_text>
Output: <generated_headline>

The encoder processes the full article (up to 512 tokens), and the decoder generates the headline using beam search (4 beams) with no-repeat ngram constraint.
